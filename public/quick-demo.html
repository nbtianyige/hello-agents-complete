📋 快速开始

  1. 安装 Python 3.10+ 和 Node.js 16+
  2. 克隆仓库：git clone https://github.com/nbtianyige/hello-agents-complete.git
  3. 配置 API Keys (DeepSeek/OpenAI + 高德地图)
  4. 启动后端：cd backend && uvicorn app.api.main:app --reload
  5. 启动前端：cd frontend && npm run dev
  6. 打开 http://localhost:5173 开始使用！
